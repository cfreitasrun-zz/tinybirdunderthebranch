= Detailed Description =

This portfolio grid plugin will show your grid, portfolio, photo gallery or album within fluid layouts to fit the entire content inside the container without stretching projects or pictures. Grid Kit portfolio grid plugin provides the most popular 3 layout schemes and modern layout styles ( square or rectangular grids, masonry, puzzle layout styles ) to show off your portfolio items or photo gallery pictures. This portfolio grid plugin comes with an stylish full screen and 5 colorbox popup viewers. There are more then 100 predefined hover effects and animations which are customizable. With this plugin you'll be able to show your project titles, descriptions, links or magnification buttons on hover actions. Grid Kit supports category filtration & pagination. It's super easy to assign a category to each portfolio item or picture. Without separately adding categories, you can just add a comma separated tags ( categories ), like you're entering different email recipients on your mail client. Grid Kit portfolio grid plugin will make your grid or portfolio social by one click. Just enable social buttons and your customers will be able to share your pictures or portfolio projects ( via Facebook, Twitter, Google Plus or Pinterest ). Grid Kit portfolio grid plugin provides very simple and user friendly layout composer where you can create your grids, portfolios or galleries and customize them. This portfolio grid plugin supports all necessary design customization options you'll need to customize the look and feel of your layouts. If the available options are not enough to fully customize your portfolio grids to fit it your needs, no problem, there is a possibility to write a native custom CSS & JS. If you’re familiar with CSS or JS you will easily customize it. Grid Kit portfolio plugin also supports short-codes, which aims to embed a portfolio or grid on any page by one click.
